# Phrasal Verbs Game
<img src="https://github.com/gmaciel10/Phrasal-Verbs-Game/blob/main/example.png?raw=true" alt="Texto Alternativo">
Descrição
O Phrasal Verbs Game é uma aplicação interativa desenhada para ajudar estudantes de inglês a aprender e praticar phrasal verbs de forma divertida e envolvente. Este jogo desafia os jogadores a associar phrasal verbs com seus significados correspondentes através de um formato de quiz dinâmico.

# Como Jogar
Ao iniciar o jogo, os jogadores são apresentados com um phrasal verb e devem escolher o significado correto dentre várias opções. Cada escolha correta acumula pontos e a progressão continua até que todos os phrasal verbs disponíveis sejam utilizados. No final, os jogadores podem reiniciar o jogo para tentar melhorar sua pontuação ou praticar phrasal verbs que encontraram mais desafiadores.

Funcionalidades
Interface Visual Atraente: A interface do jogo é colorida e amigável, tornando o aprendizado uma experiência agradável.
Fluxo de Jogo Intuitivo: O jogo é fácil de entender e jogar, adequado para estudantes de todas as idades.
Feedback Imediato: Os jogadores recebem feedback imediato sobre suas respostas, permitindo-lhes aprender e corrigir erros em tempo real.
Sistema de Pontuação: O jogo inclui um sistema de pontuação que motiva os jogadores a melhorarem seu desempenho.
Reinicio do Jogo: Os jogadores podem facilmente reiniciar o jogo para tentar alcançar uma pontuação perfeita, proporcionando uma excelente oportunidade de revisão.
Tecnologias Utilizadas
O jogo foi desenvolvido utilizando HTML, CSS e JavaScript, destacando-se pela simplicidade e eficácia. O jogo carrega os dados de um arquivo JSON, facilitando a adição ou modificação de conteúdo sem necessidade de ajustes profundos no código.
